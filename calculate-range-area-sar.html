3 Calculate Range Area: SAR

3.1 Overview

EOO values were unavailable for 17 species. For these species, total range area was estimated using 1 km² raster maps from NCC.

The workflow consisted of the following steps:

  1. Load the species’ range raster.
  2. Calculate the area of each raster cell (m²).
  3. Sum the area of all occupied (non‑NA) cells.
  4. Convert the total area to km².

Inputs
This section uses species range rasters (from NCC) located in:

  • data/raster files

3.1.1 Load Libraries

library(terra)
library(dplyr)
library(knitr)

3.2 Example: Island Blue

The code below demonstrates the range area calculation for Island Blue.

# Load raster data in for Island Blue
range_raster <- rast("data/raster files/COSEWICID_651_1KM.tif")

# Calculate the cell area in square meters
cell_areas <- cellSize(range_raster)

# Sum the area of all cells that are not NA (cells where the species is present)
# Calculates the total area of all non-NA cells by summing the individual areas
total_area_m2 <- global(cell_areas * !is.na(range_raster), "sum", na.rm=TRUE)[1,1]

# Convert the area from square meters to square kilometers
total_area_km2 <- total_area_m2 / 1e6

# Print the final area
cat("Total area of Island Blue range map:", round(total_area_km2, 2), "km²\n")
## Total area of Island Blue range map: 2799.03 km²

3.3 Full Loop

This loop goes through each of the 17 species without published EOO data. For each species the total range area is calculated and compiled into a summary table. Results were then added to data/Master_list_SAR.csv

# Set raster file locations and names for each species
species_files <- c(
  "Island Blue" = "data/raster files/COSEWICID_651_1KM.tif",
  "Purple Sanicle" = "data/raster files/COSEWICID_677_1KM.tif",
  "Vancouver Island Beggarticks" = "data/raster files/COSEWICID_688_1KM.tif",
  "Fernald's Milk-vetch" = "data/raster files/COSEWICID_4_1KM.tif",
  "Engelmann's Quillwort" = "data/raster files/COSEWICID_178_1KM.tif",
  "Gattinger's Agalinis" = "data/raster files/COSEWICID_180_1KM.tif",
  "American Ginseng" = "data/raster files/COSEWICID_217_1KM.tif",
  "Bashful Bulrush" = "data/raster files/COSEWICID_250_1KM.tif",
  "Juniper Sedge" = "data/raster files/COSEWICID_575_1KM.tif",
  "Barrens Willow" = "data/raster files/COSEWICID_678_1KM.tif",
  "Loggerhead Shrike migrans subspecies" = "data/raster files/COSEWICID_26_1KM.tif",
  "Scarlet Ammannia" = "data/raster files/COSEWICID_577_1KM.tif",
  "Riddell's Goldenrod" = "data/raster files/COSEWICID_640_1KM.tif",
  "American Badger jeffersonii subspecies (Eastern population)" = "data/raster files/COSEWICID_1230_1KM.tif",
  "American Badger jeffersonii subspecies (Western population)" = "data/raster files/COSEWICID_1229_1KM.tif",
  "Western Bumble Bee mckayi subspecies" = "data/raster files/COSEWICID_1256_1KM.tif",
  "Western Bumble Bee occidentalis subspecies" = "data/raster files/COSEWICID_1267_1KM.tif"
)

# Create an empty data frame to store results
results_df <- data.frame(Species = character(), Total_Area_km2 = numeric(), stringsAsFactors = FALSE)

# Loop over each species and calculate the total range area
for(species in names(species_files)) {
  
  # Load species raster
  range_raster <- rast(species_files[species])
  
  # Calculate the cell area in square meters
  cell_areas <- cellSize(range_raster)
  
  # Sum the area of all cells where raster is not NA
  # Calculates the total area of all valid cells by summing the individual areas
  total_area_m2 <- global(cell_areas * !is.na(range_raster), "sum", na.rm=TRUE)[1,1]
  
  # Convert the area from square meters to square kilometers
  total_area_km2 <- total_area_m2 / 1e6
  
  # Bind results to the summary table
  results_df <- rbind(results_df,
                      data.frame(Species = species,
                                 Total_Area_km2 = round(total_area_km2, 2),
                                 stringsAsFactors = FALSE))
}

# Display the results in a table
kable(results_df, caption = "Range Area for Species Missing EOO (n=17)")
